# java-filmorate
Социальная сеть с помощью, которой можно выбрать кино на основе того, какие оценки вы и ваши друзья ставите просмотренным фильмам.
Приложение реализовано с помощью:
- **Spring Boot**;
- **JdbcTemplate**;
- БД - **H2**;
- **JUnit 5**;
